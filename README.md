React Native Posts App - ExactSpace Mobile Developer Assessment

## Features Implemented (All Mandatory + Bonuses):

**Mandatory Requirements:**
✓ Fetch posts from https://jsonplaceholder.typicode.com/posts
✓ FlatList displaying title and body using reusable PostCard component
✓ Search input filtering posts by title (case-insensitive, live typing)
✓ AsyncStorage saving search text with auto-restore on app restart
✓ Error handling: "Unable to fetch posts. Check your network connection."
✓ Empty state: "No posts found" when no search results

**Bonus Features Implemented:**
✓ Loading indicators + skeleton loaders
✓ Pull-to-refresh functionality
✓ Clean folder architecture (components, hooks, services, types)
✓ TypeScript with full type safety
✓ Custom hooks (usePosts, useDebounce)
✓ Production-ready APK generated
